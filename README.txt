
SteelCraft Clean Starter (RTL)
------------------------------
1) أنشئ مستودع جديد في GitHub (عام).
2) ارفع كل الملفات كما هي (بدون ZIP داخل ZIP).
3) Settings → Pages → Source: Deploy from a branch, Branch: main, Folder: /(root).
4) افتح: https://USERNAME.github.io/REPO-NAME/
- الروابط داخلية نسبية ./ لتفادي مشاكل الدومين.
- عدّل رقم الواتساب داخل app.js (const phone).
- عدّل المنتجات داخل data/products.json.
